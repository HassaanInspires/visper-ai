Legal & Compliance

Privacy Policy & Data Guarantee.

Effective Date: July 23, 2026 | Extension Version: 2.5.0

1. Single Purpose & Overview

Visper AI is an autonomous AI browser copilot built strictly in compliance with Google Chrome's Manifest V3 Developer Program Policies. Its single purpose is to provide users with a side panel AI assistant that automates e-commerce size selection and checkout forms, synthesizes web research into comparison tables, indexes local PDF documents via WASM vector embeddings, and generates bilingual YouTube transcript summaries.

2. Zero-Telemetry & Local-First Disclosures

  • Zero Telemetry / Trackers: Visper AI contains 0 third-party tracking scripts, 0 analytics SDKs, 0 advertising networks, and 0 external telemetry servers.
  • Local Storage (chrome.storage.local): All API keys (OpenAI, Anthropic, Gemini, Groq, DeepSeek) and settings remain stored strictly on your local device sandbox.
  • Local Vector Database (IndexedDB & WASM): Document vector search embeddings are calculated offline inside your browser sandbox (@xenova/transformers) and stored in local IndexedDB.

3. Permission Justification Matrix for Chrome Review Team

Permission Purpose & Necessity
sidePanel Renders the primary assistant chat interface side-by-side with active browser tabs.
activeTab Extracts text and DOM metadata from the active tab only when requested by the user.
tabs Opens research result tabs and allows navigation during user-initiated web research tasks.
scripting Executes DOM action scripts (e.g., size selection, coupon testing) authorized by the user.
storage Saves local user preferences, theme options, API keys, and local chat history.
offscreen Runs client-side WASM text vectorization for document RAG without freezing the main UI.
<all_urls> Enables autonomous web assistance across any user-visited website during active sessions.
