Privacy Policy for Visper AI

Effective Date: July 22, 2026 | Extension Version: 2.5.0

← Back to Home

1. Single Purpose & Overview

Visper AI is an autonomous AI browser assistant and copilot extension designed solely to enhance web productivity. Its single purpose is to assist users with web content interaction, e-commerce shopping automation, web research synthesis, document vector search, and YouTube transcript summaries.

2. Data Collection & Privacy Practices

A. Data Stored 100% Locally

  • API Keys & Settings: User API keys (OpenAI, Anthropic, Gemini, Groq, DeepSeek) and extension settings are stored locally on the user's browser via chrome.storage.local.
  • Local Semantic Memory (RAG): Past chat history and uploaded PDF document chunks are vectorized locally using WebAssembly (Transformers.js ONNX) and stored locally in IndexedDB.
  • No Telemetry or Third-Party Tracking: Visper AI does NOT use tracking cookies, analytics SDKs, or external telemetry tracking servers.

B. Network Data Transmission

When a user submits a prompt, text queries and relevant page context are sent directly to the user-selected AI model provider API (e.g. OpenAI, Anthropic, Google Gemini, Groq, DeepSeek) or processed completely offline via Chrome Gemini Nano. All network requests occur via encrypted HTTPS directly between the user's browser and the AI provider with no middleman server.

3. Permissions Justification

sidePanel: Displays assistant UI alongside active web pages.
activeTab: Extracts page context when requested by the user.
tabs: Opens new research tabs during multi-step tasks.
scripting: Executes DOM action heuristics authorized by user.
storage: Saves local user preferences and chat history.
offscreen: Computes local WASM text embeddings for vector RAG.

4. Data Erasure & Control

Users retain 100% control of their data. All local chat histories and vector databases can be erased at any time from the extension settings panel. Uninstalling the extension permanently removes all stored data from the browser.

For privacy inquiries, please visit our Support Page or official GitHub Repository.
