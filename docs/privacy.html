Privacy Policy for Visper AI

Last Updated: July 23, 2026 | Extension Version: 2.5.0

← Back to Home

1. Single Purpose & Overview

Visper AI is an autonomous AI browser assistant extension built in compliance with Google Chrome's Manifest V3 standards. Its single purpose is to provide users with an intelligent browser side panel assistant that helps with e-commerce size selection and checkout automation, web research table compilation, local PDF document vector search, and bilingual YouTube video transcript summaries.

2. Zero-Telemetry & Local Data Storage Disclosures

Visper AI operates on a local-first data privacy architecture. We prioritize user data security above all else:

  • No Telemetry or Tracking: Visper AI contains 0 third-party tracking scripts, 0 analytics SDKs, 0 advertising networks, and 0 external telemetry servers.
  • Local Storage (chrome.storage.local): Your API keys (OpenAI, Anthropic, Gemini, Groq, DeepSeek) and custom settings are stored exclusively inside your browser's local sandbox storage. They are never sent to us or any central server.
  • Local Vector Database (IndexedDB & WASM): When using document vector search (Local RAG), document embeddings are generated entirely offline inside your browser using WebAssembly (@xenova/transformers) and stored in local IndexedDB.

3. Transmitted Data & Network Security

When you submit a prompt or request an action, text queries and page content selected by you are transmitted via direct, encrypted HTTPS requests to the third-party AI model provider you selected (e.g. OpenAI, Anthropic, Google Gemini, Groq, or DeepSeek).

Alternatively, if you choose Local Gemini Nano, all completions execute 100% offline within your device with zero network transmission.

4. Chrome Extension Permission Justifications Matrix

The following table outlines the exact permissions requested in manifest.json and their strict necessity for the extension's single purpose:

Permission Purpose & Necessity
sidePanel Renders the primary assistant chat interface side-by-side with active browser tabs.
activeTab Extracts text and DOM metadata from the active tab only when requested by the user.
tabs Opens research result tabs and allows navigation during user-initiated web research tasks.
scripting Executes DOM action scripts (e.g., size selection, coupon testing) authorized by the user.
storage Saves local user preferences, theme options, API keys, and local chat history.
offscreen Runs client-side WASM text vectorization for document RAG without freezing the main UI.
<all_urls> Enables autonomous web assistance across any user-visited website during active sessions.

5. Data Retention & User Rights (GDPR / CCPA)

You retain 100% ownership and control over your data. You may clear your chat history, vector indexes, or API keys at any time from the extension's settings panel. Uninstalling Visper AI permanently deletes all local storage items from your device.
