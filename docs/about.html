🎯 Our Single Purpose

Visper AI is designed solely to provide users with an autonomous, context-aware AI side panel assistant inside Google Chrome. It automates shopping size selection and checkout forms, synthesizes web research into comparison tables, indexes local PDF documents via WebAssembly vector embeddings, and generates bilingual YouTube video summaries in English and Urdu.

🔒
100% Local RAG

Document indexes and chat histories stay on your machine in local IndexedDB.

⚡
Zero Complexity

Use your favorite model API key or free offline Chrome Gemini Nano.

🛡️
Manifest V3 Secure

Strict compliance with Google Chrome Manifest V3 security requirements.

⚙️ Direct Encrypted Network Architecture

Visper AI contains zero middleman servers. When you prompt the AI, your browser communicates directly via encrypted HTTPS with your selected AI provider (OpenAI, Anthropic, Google Gemini, Groq, DeepSeek). If you select Local Gemini Nano, all computation executes completely offline.
